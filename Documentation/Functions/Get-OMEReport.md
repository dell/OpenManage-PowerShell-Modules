# Get-OMEReport

## SYNOPSIS
Script to retrieve the list of pre-defined reports

## SYNTAX

```
Get-OMEReport [<CommonParameters>]
```

## DESCRIPTION
This script uses the OME REST API.
Note that the credentials entered are not stored to disk.

## EXAMPLES

### EXAMPLE 1
```
Get-OMEReport | Format-Table
```
