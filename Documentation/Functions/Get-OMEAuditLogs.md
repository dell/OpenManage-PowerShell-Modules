# Get-OMEAuditLogs

## SYNOPSIS
Retrieves the audit logs from a target OME instance

## SYNTAX

```
Get-OMEAuditLogs [<CommonParameters>]
```

## DESCRIPTION
It performs X-Auth with basic authentication.
Note: Credentials are not stored on disk.

## EXAMPLES

### EXAMPLE 1
```
Get-OMEAuditLogs | Format-Table
```
